そもそもメモリ半導体って何？
CPUやGPUの横でデータを覚えて渡す係
⚡ このセクションの3秒まとめ
  • GPUが計算係なら、メモリは必要データをすぐ渡す係。
  • AIでは大量データを超高速で動かすHBMが重要。
  • メモリは供給過多になると価格が急落しやすい。

AIチップを超速い料理人だと思ってください。材料を運ぶ係が遅ければ、料理人は待つしかない。HBMは材料を一気に運ぶ超高速ベルトコンベアです。

つまり：AI計算が増えるほど、速くて大容量のメモリ価値が上がりやすい。

DRAMDynamic RAM⌄

①定義：作業中のデータを一時保存する高速メモリ。
②なぜ重要：PC・スマホ・サーバー全部に必要。
③MUでは：売上と利益の大黒柱。

NANDFlash Memory⌄

①定義：電源を切ってもデータが残るメモリ。
②なぜ重要：SSDやスマホ保存領域に使う。
③MUでは：データセンター向けSSDが成長分野。

HBMHigh Bandwidth Memory⌄

①定義：DRAMを縦に積み、高速で大量データを運ぶメモリ。
②なぜ重要：AI GPUの性能を引き出す。
③MUでは：利益成長の中心テーマ。

メモリサイクルMemory Cycle⌄

①定義：不足→値上がり→増産→余る→値下がりを繰り返すこと。
②なぜ重要：利益が大きく上下する。
③MUでは：株価を見るうえで絶対に外せない。

Micronは何を売ってる？
DRAM・HBM・NANDをざっくり攻略

  • NANDはSSDなどの長期保存庫。

HBM3E / HBM4AI Memory Generations⌄

世代が進むほど帯域・容量・電力効率が改善。MicronはHBM3EとHBM4の供給拡大が重要。

データセンターDRAMServer DRAM⌄

AIサーバーは一般PCより大量のメモリを使う。AI設備投資の恩恵はHBMだけではない。

Enterprise SSDEnterprise Solid State Drive⌄

4
AI時代の本丸、HBMは何が難しい？
速さだけじゃない。積層・歩留まり・認定が勝負
⚡ このセクションの3秒まとめ
  • HBMはDRAMを縦積みするため製造が難しい。
  • 性能だけでなく顧客の厳しい認定を通る必要がある。
  • 歩留まりが悪いと、売れても供給量と利益が伸びにくい。
  • 🏗️積層：複数のDRAMを縦に重ねて超高速化。
  • 🎯歩留まり：良品率が低いとコストが跳ねる。
  • 🤝顧客認定：AIアクセラレーターとの組み合わせで性能・品質を確認。
TSVThrough-Silicon Via⌄

シリコンを縦に貫く電極。積層したメモリ同士を高速接続する。

歩留まりYield⌄

作ったチップのうち良品になる割合。HBMの利益率を左右する。

顧客認定Qualification⌄

顧客側で性能・信頼性を検証する工程。採用まで時間がかかる。

先端パッケージAdvanced Packaging⌄

GPUとHBMを近くに配置する高度な実装技術。供給能力がAIチップ全体のボトルネックになりうる。

設備投資Capex⌄

工場や製造装置への投資。増やせば供給は伸びるが、将来の供給過多リスクも生む。
